7 Penjelasan Gerbang Logic Gate
1. AND
Pada gerbang AND output akan bernilai 1 jika inputnya bernilai 1 semua. Jika salah satu 
bernilai 0 atau semuanya bernilai 0, maka outputnya akan 0.
![image alt](https://github.com/halimraka/sisdihdjawa/blob/f5adc3226b5b8217fab299a4406aded4cfa12002/AND.png)
2. NAND
Pada gerbang NAND ini merupakan gabungan antara gerbang AND dan NOT serta 
kebalikan dari gerbang AND. Output akan bernilai 0 jika inputnya bernilai 1 semuanya.

3. OR
Pada gerbang OR, output akan bernilai 1 jika inputnya bernilai 1 semua atau terdapat salah 
satu yang bernilai 1. Jika inputnya bernilai 0 semua, maka outputnya akan bernilai 0.

4. NOR
Pada gerbang NOR ini merupakan gabungan antara gerbang OR dan NOT serta kebalikan 
dari gerbang OR. Output akan bernilai 1 jika inputnya bernilai 0 semua.

5. EX-OR
Pada gerbang EX-OR, output akan bernilai 1 jika inputnya berbeda. Jika inputnya sama, 
baik bernilai 1 semua atau 0 semua, maka outputnya akan bernilai 0.

6. EX-NOR
Pada gerbang EX-NOR merupakan kebalikan dari gerbang EX-OR. Output akan bernilai 
1 jika inputnya sama, baik bernilai 1 semua atau 0 semua. Jika terdapat input yang berbeda 
maka outputnya akan bernilai 0.

7. NOT
Pada gerbang NOT merupakan gerbang inverter, di mana gerbang ini memiliki logika yang 
membalikan keadaan pada input, jika input bernilai 1 maka output akan bernilai 0, 
begitupun sebaliknya. Gerbang ini hanya memiliki satu input dan satu output.
